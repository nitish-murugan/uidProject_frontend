🐛 TROUBLESHOOTING:

   MongoDB not running?
   → net start MongoDB
   
   Port already in use?
   → Check .env file or kill process
   
   Module not found?
   → npm install (in both root and server folders)
   
   CORS errors?
   → Ensure backend is on port 5000
